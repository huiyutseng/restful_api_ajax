# Ollama

## 安裝 Ollama
- [Download Ollama](https://ollama.com/download/windows)

## 使用 Ollama 下載模型：
  - [Gemma4](https://ollama.com/library/gemma4)
    - `ollama run gemma4:e2b`
  - [BGE-M3](https://ollama.com/library/bge-m3)
    - `ollama pull bge-m3`
